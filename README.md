## About

Python package for harvesting records from the UNT Digital Library via OAI plus tools for scraping full text from the PDFs.

## Installation

Clone the repository.

Install the `untdl_harvest` package.

```bash
$ cd untdl-harvest
$ python -m pip install .
```
